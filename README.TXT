# Tic Tac Toe Game 🎮

A simple and interactive Tic Tac Toe game built using **React**. This project is perfect for beginners looking to understand component-based UI, state management, and game logic in React.

---

## 🛠️ Features

- Classic 2-player Tic Tac Toe gameplay
- Game status display (winner/draw)
- Restart functionality

---

## 🚀 Getting Started

### Prerequisites

- **Node.js** and **npm** installed
- **Git Bash** (for Windows users)

### Clone and Setup

Follow these steps in **Git Bash**:

```bash
# Clone the repository
git clone https://github.com/ArkoKhan/React_Tic_Tac_Toe_game.git

# Navigate into the project directory
cd tic-tac-toe-react

# Install project dependencies
npm install

# Start the development server
npm run dev
